# Ganache Transaction Viewing Guide

Since Ganache doesn't have a web-based block explorer like Etherscan, here's how to view your blockchain transactions:

## Method 1: Ganache GUI
1. Open the Ganache application
2. Go to the "Transactions" tab
3. You'll see all transactions with their hashes, block numbers, and gas costs
4. Click on any transaction to see detailed information

## Method 2: Using Web3 Console (Advanced)
You can connect to Ganache using Web3 to query transaction details:

```javascript
// Connect to Ganache
const Web3 = require('web3');
const web3 = new Web3('http://localhost:7545');

// Get transaction details by hash
async function getTransactionDetails(txHash) {
    const tx = await web3.eth.getTransaction(txHash);
    const receipt = await web3.eth.getTransactionReceipt(txHash);
    console.log('Transaction:', tx);
    console.log('Receipt:', receipt);
}

// Example usage
getTransactionDetails('0x5148a1bb2dc06b8e2c6bd832084edc24ccd5276eef8b66650ff940a83e7ee559');
```

## Transaction Information Available:
- **Transaction Hash**: Unique identifier for the transaction
- **Block Number**: The block containing the transaction
- **Gas Used**: Computational cost of the transaction
- **From/To Addresses**: Sender and recipient addresses
- **Contract Events**: Events emitted by the smart contract
- **Status**: Success (1) or Failed (0)
